Common Assembly Instructions for Malware Analysis
Instruction Description
MOV Moves data from one location to another (e.g., from a register to memory).
ADD Adds two values together and stores the result.
SUB Subtracts one value from another.
PUSH Stores a value on the stack (temporary storage).
POP Removes and retrieves the last value from the stack.
CALL Jumps to a function or procedure and saves the return address.
RET Returns from a function to the saved return address.
JMP Unconditional jump to another part of the program.
JE/JZ Jumps if equal (or zero); typically after a comparison.
JNE/JNZ Jumps if not equal (or not zero); after a comparison.
INT Triggers an interrupt, often used for system calls.
INC Increases the value of a register or memory location by 1.
DEC Decreases the value of a register or memory location by 1.
OR Performs a bitwise OR operation on two values.
AND Performs a bitwise AND operation on two values.
XOR Performs a bitwise XOR (exclusive OR), often used to zero out registers.
SHL Shifts the bits of a value to the left (multiplies by powers of 2).
SHR Shifts the bits of a value to the right (divides by powers of 2).
LEA Loads the effective address of a memory operand into a register.
NOP No operation; does nothing. Often used to fill space.
Reverse Engineering Tools
Tool Description
Ghidra A free, open-source reverse engineering tool developed by NSA for analyzing executable files.
IDA Pro Interactive Disassembler widely used for disassembling and reverse engineering binaries.
Binary Ninja A reverse engineering platform for analyzing binary code, offering both disassembly and decompilation.
Radare2 A set of tools for reverse engineering and analyzing binaries, focusing on flexibility and performance.
x64dbg An open-source debugger for analyzing 64-bit executables.
Document Analysis Tools
Tool Description
olevba Tool to analyze VBA macros in Office documents for potential malicious behavior.
oleid Tool to detect suspicious characteristics in OLE files like embedded objects and macros.
mraptor Tool to detect malicious features in Microsoft Office documents.
pdfid Identifies potentially malicious PDFs by analyzing objects and suspicious features.
pdf-parser PDF analysis tool used to extract, analyze, and parse PDF contents for malicious elements.
PyMuPDF A Python binding for the MuPDF library, used to work with PDF files for analysis.
EXE Analysis Tools
Tool Description
PEiD Identifies packed executables and compilers for Windows PE files.
PEview Allows for viewing the structure and contents of PE files.
CFF Explorer A tool for working with PE32 and PE64 files to view headers, sections, and resources.
ELF Analysis Tools
Tool Description
readelf Displays information about ELF files and their sections, symbols, and segments.
objdump Displays various aspects of object files and ELF executables, including disassembly.
eu-readelf A modern implementation of readelf for examining ELF file contents.
APK Analysis Tools
Tool Description
APKTool Tool for decompiling, disassembling, and modifying Android APK files.
JD-GUI Java decompiler for viewing the source code of APKs and other Java-based applications.
androguard A Python tool for analyzing Android applications and APK contents.
